  INTER-SERVICE EVENT MAP
  ───────────────────────
  wallet-service publishes:
    WALLET_GENERATED → dashboard-service
    WALLET_FUNDED → dashboard-service
    SWAP_COMPLETED → contract-service + dashboard-service
    SWAP_FAILED → dashboard-service
    RETURN_COMPLETED → dashboard-service

  contract-service publishes:
    CONTRACT_DEPLOYED → dashboard-service
    CONTRACT_FUNDED → dashboard-service
    AIRDROP_STARTED → dashboard-service
    AIRDROP_PAUSED → dashboard-service
    AIRDROP_STOPPED → dashboard-service
    WITHDRAW_COMPLETED → wallet-service + dashboard-service

  dashboard-service consumes ALL events → logs + state updates + Socket.io
